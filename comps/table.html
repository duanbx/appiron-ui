# Table 基础表格

基于 element-plus 二次封装

# 基础表格

基础的表格展示用法。

data 定义表格数据 tableColumn 定义表格列数组
查看源代码
<template>
  <a-table :data="data.data" :tableColumn="data.tableColumn"></a-table>
</template>
<script lang="ts" setup>
import { ATable } from "appiron-ui"
import { reactive } from "vue"
const data = reactive({
  data: [
    {
      date: "2016-05-03",
      name: "Tom",
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-02",
      name: "Tom",
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-04",
      name: "Tom",
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-01",
      name: "Tom",
      address: "No. 189, Grove St, Los Angeles",
    },
  ],
  tableColumn: [
    { key: "date", label: "日期" },
    { key: "name", label: "姓名" },
    { key: "address", label: "地址" },
  ],
})
</script>

# 显示勾选和序列的表格

勾选和序列的表格展示用法。

传入 showIndex 可显示序号，传入 checkList 可显示勾选，如果 checkList 是数组为多选，对象为单选
查看源代码
<template>
  <el-space wrap direction="vertical" :size="30">
    <a-table
      :data="data.data"
      :tableColumn="data.tableColumn"
      showIndex
      v-model:checkList="data.checkList"
    ></a-table>
    <a-table
      :data="data.data"
      :tableColumn="data.tableColumn"
      showIndex
      v-model:checkList="data.checkObject"
    ></a-table>
  </el-space>
</template>
<script lang="ts" setup>
// import { ATable } from "appiron-ui"
import { reactive } from "vue"
const data = reactive({
  data: [
    {
      date: "2016-05-03",
      name: "Tom",
      id: 1,
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-02",
      name: "Tom",
      id: 2,
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-04",
      name: "Tom",
      id: 3,
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-01",
      name: "Tom",
      id: 4,
      address: "No. 189, Grove St, Los Angeles",
    },
  ],
  tableColumn: [
    { key: "date", label: "日期" },
    { key: "name", label: "姓名" },
    { key: "address", label: "地址" },
  ],
  checkList: [
    {
      date: "2016-05-04",
      name: "Tom",
      id: 3,
      address: "No. 189, Grove St, Los Angeles",
    },
  ],
  checkObject: {
    date: "2016-05-04",
    name: "Tom",
    id: 3,
    address: "No. 189, Grove St, Los Angeles",
  },
})
</script>

# 自定义列

自定义列表格展示用法。

需要指定 slot 为 true name 为对应列的 key 加上 Col （nameCol）
查看源代码
<template>
  <a-table showIndex :data="data.data" :tableColumn="data.tableColumn">
    <template #nameCol> 自定义姓名 </template>
  </a-table>
</template>
<script lang="ts" setup>
import { reactive } from "vue"
const data = reactive({
  data: [
    {
      date: "2016-05-03",
      name: "Tom",
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-02",
      name: "Tom",
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-04",
      name: "Tom",
      address: "No. 189, Grove St, Los Angeles",
    },
    {
      date: "2016-05-01",
      name: "Tom",
      address: "No. 189, Grove St, Los Angeles",
    },
  ],
  tableColumn: [
    { key: "date", label: "日期" },
    { key: "name", label: "姓名", slot: true },
    { key: "address", label: "地址" },
  ],
})
</script>

# Table 属性

属性说明类型可选值默认值
data表格数据unknow[]-[]
table-column表格列数组Column[]-[]
row-key表格列唯一 key,启用 checkList 必须传，不能重复String-id
check-list默认勾选的数据Array | ObjectUndefined
reserve-selection分页的时候是否记录所有分页的勾选数据Booleanfalse

| show-index | 是否显示序号 | Boolean | - | false |
